# Northern Territory Open Data (data-nt-gov-au)

Northern Territory Open Data is a territorial government open-data portal for Australia running **CKAN**, exposing a standardized catalog API over ~1,071 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-nt-gov-au/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=data-nt-gov-au-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.11.5

## API
- **Northern Territory Open Data CKAN Action API** — base `https://data.nt.gov.au/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://data.nt.gov.au)
- Live endpoint verified: `https://data.nt.gov.au/api/3/action/status_show` · ~1,071 datasets
